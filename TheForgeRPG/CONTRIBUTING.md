# 🤝 Contributing to FORGE RPG

Thank you for your interest in making FORGE RPG better for students and social workers everywhere!

## Ways to Contribute

**You don't need to know how to code.** The most valuable contributions come from practitioners in the field.

### 🏫 I'm a School Social Worker or Counselor
- Share your experience running the program (create a GitHub Issue labeled "Practitioner Feedback")
- Submit modifications that worked for specific populations
- Report any safety concerns or protocol gaps

### 🎲 I'm a TTRPG Designer
- Create new adventure modules using the `ADVENTURE_MODULE_TEMPLATE.md`
- Improve the core rules for therapeutic clarity
- Develop new character archetypes

### 🔬 I'm a Researcher
- Conduct a pilot study and share results (contact maintainers first)
- Update the Research Foundation with new publications
- Develop validated assessment tools to pair with the program

### 🎨 I'm an Artist or Designer
- Create accessible, culturally diverse character art
- Design printable character sheets and handouts
- Create visual aids for the emotion wheel and rule summary

### 🌍 I Speak Another Language
- Translate materials into Spanish, Mandarin, or other languages
- Review existing translations for cultural appropriateness
- Adapt materials for non-US educational contexts

## Submission Process

1. Fork the repository
2. Create a branch: `git checkout -b your-contribution-name`
3. Make your changes
4. Submit a Pull Request with a clear description
5. For new modules: include the therapeutic rationale and research basis

## Standards

All contributions must:
- Be licensed CC BY-SA 4.0
- Include a research basis for therapeutic claims
- Pass review by at least one licensed school mental health professional
- Be trauma-informed and culturally responsive

## Code of Conduct

This project is dedicated to students who need support. All contributors are expected to maintain professionalism and respect in all interactions.
